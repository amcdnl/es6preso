# ES6 JavaScript Presentation

Topics:

- Module Loaders
- Using it today!

Visit [Slides.com](http://slides.com/austinmcdaniel/deck) for the presentation slides.

## Requirements

- NodesJS & NPM
- [Bower](http://bower.io/)
- [GulpJS](http://gulpjs.com/)
- [JSPM](http://jspm.io/)

## Install

1. `npm install`
2. `gulp watch serve`
3. Browse to `http://localhost:9000`
